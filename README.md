# File Renamer Script

This is a simple Python Script that renames all files inside a specified folder by removing dashes (-) from their filenames.
The program is created using Object-Oriented Programming (OOP) principles and uses the `pathlib` library for handling file paths.

Example:

Before:
12-121-241.txt  
my-file-name.docx  

After:
12121241.txt  
myfilename.docx  

---

##  Technologies Used

- Python 
- pathlib library
